# EE243A 計算機概論(二) 期中專題：C 語言 2048 遊戲

## 🎓 課程資訊
* **學校/系級**：電機一甲
* **指導教授**：王佳盈 老師
* **開發者**：魏銘宏

## 🎮 遊戲簡介
本專案使用純 C 語言實作經典遊戲 2048。透過控制台 (Console) 介面與玩家互動，並加入顏色顯示以提升遊戲體驗。

## 🛠️ 技術特色
1. **矩陣旋轉演算法**：利用線性代數旋轉矩陣的方式，僅用一個 `move_left` 函數即可處理四個方向的移動，大幅降低代碼重複率。
2. **ANSI Color UI**：在終端機實作動態顏色輸出，根據數值大小變換顏色。
3. **即時按鍵偵測**：引用 `conio.h` 實作非阻塞式輸入，玩家無需按 Enter 即可操作。

## 🚀 如何執行
1. 使用 GCC 編譯：`gcc main.c -o 2048`
2. 執行程式：`./2048`
3. 操作說明：
   - **W/A/S/D**：上下左右移動
   - **Q**：結束遊戲
## 心得
在開發過程中發現，使用 VS Code 內建的 Debug 箭頭執行時，由於 Debugger 對執行檔的鎖定（Locking）以及對 _getch() 緩衝區處理的限制，導致輸入無反應或編譯權限衝突。

解決方案：
最終決定採用「分離編譯與執行」的策略。透過手動在 Terminal 呼叫二進位執行檔 (.\game.exe)，避開了 IDE 的監控層，成功實現了低延遲的遊戲操作體驗。
